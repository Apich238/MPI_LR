Решение системы линейных уравнений методом Гаусса с выбором главного элемента в матрице

ввод A, b
прямой ход метода гаусса
обратный ход метода гаусса
x=b'
проверка

прямой ход метода гаусса
  для всех строк (пока они все не попадут в список обработанных элементов)
    выбор преобладающего элемента --> Aij
    вычитание строки i из остальной матрицы кроме i, с соответствующими коэффициентами, чтобы занулить всё остальное в этом j столбце
    добавление (i,j) в список обработанных главных элементов

обратный ход метода гаусса
  для всех элементов (i,j) из списка обработанных, начиная с конца
    вычесть строку i из остальных, с соответствующими коэффициентами

проверка
  |Ax-b|<eps --> правильно, иначе - неправильно
